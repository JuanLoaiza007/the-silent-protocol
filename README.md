# Skyward Gate

A 3D game project developed using the Godot game engine.

## Requirements

- **Godot Version**: 4.5.1-stable
  Download from: [https://godotengine.org/download/archive/4.5.1-stable/](https://godotengine.org/download/archive/4.5.1-stable/)
- **Scripting Language**: GDScript (standard)

## Project Structure

- `addons/` - Custom addons and plugins for the project.
- `assets/` - Game assets including audio, images, models, and textures.
- `characters/` - Character-related scenes, scripts, and assets (e.g., player, villains).
- `game_components/` - Reusable game components and mechanics.
- `game_world/` - Elements related to the game world and environment.
- `levels/` - Individual level scenes and configurations.
- `main/` - Main entry point scenes and scripts for the game.
- `scripts/` - Shared scripts and utilities.
- `ui/` - User interface elements and scenes.

Each directory should contains a `README.md` file with more detailed information about its contents.
